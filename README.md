# tui CLI

A CLI to generate commonly used React patterns and file structures.

## Installation

Clone the repo

```
git clone https://github.com/jacksonhardaker/tui.git

cd tui

```

Install dependencies
```
npm i
```

Link with NPM (so that you can use it from anywhere)
```
npm link
```

### Usage

Basic usage
```
tui help
tui --help
tui -h
```
